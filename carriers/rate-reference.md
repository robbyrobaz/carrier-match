## Level/Preferred vs GI Cost Comparison (Why Qualification Matters)

A healthy 65-year-old female seeking $15,000 coverage:
- **Level (Preferred)**: ~$45–55/month (Mutual of Omaha, Americo)
- **Graded**: ~$60–75/month
- **GI**: ~$95–105/month

The GI penalty is roughly **2x the cost** of a Level policy for the same coverage.
This is why finding the highest tier the client qualifies for is so important.

## Average Funeral Cost Context (Why This Coverage Amount Matters)

Help agents frame the conversation with prospects:

| Service | Average Cost (2024) |
|---------|---------------------|
| Traditional burial with funeral | $7,000–$12,000 |
| Cremation with service | $2,000–$5,000 |
| Direct cremation (no service) | $700–$1,500 |
| Cemetery plot | $1,000–$4,000 |
| Headstone | $1,000–$3,000 |
| **Total (burial + plot + stone)** | **$10,000–$20,000** |
| **Total (cremation + service + urn)** | **$3,000–$8,000** |

**Recommended coverage range**: $10,000–$15,000 covers most burial scenarios.
$20,000–$25,000 provides a buffer for inflation and additional expenses.
